## Vendored Decoder Binaries

The following decoder/transcoder libraries are vendored under `public/decoders/` so that AGF can run offline / air-gapped and so that compressed-mesh / compressed-texture assets do not depend on third-party CDNs at runtime. Each subdirectory keeps its upstream README; the licenses below are reproduced here for top-level visibility.

### Draco (`public/decoders/draco/`)

- Upstream: https://github.com/google/draco
- Authors: Google LLC
- License: Apache-2.0 (see https://github.com/google/draco/blob/main/LICENSE)
- Files: `draco_decoder.js`, `draco_decoder.wasm`, `draco_wasm_wrapper.js`
- Purpose: Decompress `KHR_draco_mesh_compression`-encoded glTF meshes at runtime.

### Basis Universal (`public/decoders/basis/`)

- Upstream: https://github.com/BinomialLLC/basis_universal
- Authors: Binomial LLC
- License: Apache-2.0 (see https://github.com/BinomialLLC/basis_universal/blob/master/LICENSE)
- Files: `basis_transcoder.js`, `basis_transcoder.wasm`
- Purpose: Transcode `.basis` / `.ktx2` GPU-compressed textures at runtime.

## Beacon World Assets

`examples/beacon-world/` ships procedural assets generated by scripts in `examples/beacon-world/assets/_sources/`. Source metadata for any imported third-party asset is tracked under `asset-sources.json` in that directory.

If an example introduces a CC0 / public-domain or other open-licensed asset, the asset-sources entry must record provenance and license. Contributors must not commit assets without provenance metadata.
